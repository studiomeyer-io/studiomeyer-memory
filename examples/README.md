# StudioMeyer Memory — Examples

Code examples for integrating [StudioMeyer Memory](https://memory.studiomeyer.io) into your AI agents.

## Examples

| File | Language | Description |
|------|----------|-------------|
| [01-basic-usage.py](01-basic-usage.py) | Python | Sign up, store memories, search, create entities |
| [02-langchain-integration.py](02-langchain-integration.py) | Python | LangChain tools with persistent memory |
| [03-crewai-integration.py](03-crewai-integration.py) | Python | CrewAI multi-agent shared memory |
| [04-typescript-basic.ts](04-typescript-basic.ts) | TypeScript | Basic REST API usage with fetch |
| [05-import-chatgpt.py](05-import-chatgpt.py) | Python | Import ChatGPT history |
| [06-sdk-usage.py](06-sdk-usage.py) | Python | Official SDK usage (requires `pip install nex-memory`) |

## Quick Start

```bash
export NEX_API_KEY="nex_your_key"
python 01-basic-usage.py
```
